# gc_odoo_analityc_filter

Filtro según cuentas analíticas en facturas para Odoo 16

## Descripción

Este módulo permite filtrar las cuentas analíticas disponibles en las facturas, facilitando la selección y mejorando la experiencia del usuario al trabajar con contabilidad analítica en Odoo.

## Características

- Filtra las cuentas analíticas que pueden seleccionarse en las facturas de cliente y proveedor.

## Instalación

1. Copiar el módulo a la carpeta de addons de Odoo.
2. Actualizar la lista de aplicaciones.
3. Instalar el módulo **Filtro segun cuentas analiticas en facturas** desde el backend de Odoo.
