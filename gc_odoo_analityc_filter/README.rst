gc_odoo_analityc_filter
=======================

Filtro según cuentas analíticas en facturas para Odoo 16.

**Tabla de contenidos**

.. contents::
   :local:

Descripción
-----------

Este módulo permite filtrar las cuentas analíticas disponibles en las facturas,
facilitando la selección y mejorando la experiencia del usuario al trabajar con
contabilidad analítica en Odoo.

Características
-----------------

* Filtra las cuentas analíticas que pueden seleccionarse en las facturas de cliente y proveedor.

Instalación
------------

#. Copiar el módulo a la carpeta de addons de Odoo.
#. Actualizar la lista de aplicaciones.
#. Instalar el módulo *Filtro segun cuentas analiticas en facturas* desde el backend de Odoo.
