l10n_ar_default_service_dates
=============================

Sets default values for service dates on customer invoices.

**Table of contents**

.. contents::
   :local:

Description
-----------

This module sets default values for the fields ``l10n_ar_afip_service_start`` and ``l10n_ar_afip_service_end`` on customer invoices (``account.move``) to the current date.
